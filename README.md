# Accordion

A lightweight, pure CSS accordion component built with HTML and CSS. Uses hidden radio buttons and CSS transitions for expand/collapse functionality — no JavaScript required.

## Features

- Pure CSS accordion (no JS)
- Neumorphic design
- Smooth expand/collapse transitions
- Radio-button behavior (one section open at a time)

## Usage

Open `index.html` in any modern browser.

## Files

- `index.html` — Accordion markup
- `style.css` — Styling and animations
